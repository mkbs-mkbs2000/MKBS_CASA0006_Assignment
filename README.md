# MKBS_CASA0006_Assignment
Repository storing ipynb file and associated data/py files to meet the coursework requirements for CASA0006: Data Science for Spatial Systems.

This coursework assignment aims to build a model that predicts cycling casualty severity for accidents that happened between 2020 and 2024 based on environmental, infrastructural and collision-area socioeconomic deprivation features.

The central file is the .ipynb file, with references to data sources in the data folder and an eda_plots_complete.py file that has the source code specific for the exploratory data visualisation section.
